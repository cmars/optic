To initialize Optic with your repository, run:

```bash
api init
```

```bash title="Output"
../repos/todo-js
Is this your API's root directory? (yes/no): yes
What is this API named?: todo-js
[optic] Added Optic configuration to .../todo-js/optic.yml
```

The initialization process creates an `optic.yml` file in your project's root folder. Read on to learn how to modify this `optic.yml` file to integrate with your project.
