### Validating your SDK integration

After setting up your integration, we recommend passing some traffic through Optic to validate it is set up correctly. For the best results, you should plan to send at least 5 requests to your application. This helps assure that multiple responses are coming back as expected. Start up your project with Optic, using the `--verbose` flag. This will give you some additional troubleshooting information for your run:

```
start an exec session, look at the output
```
